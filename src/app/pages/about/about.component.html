What I've learned

One of the key concepts I learned was how to implement two-way data binding in Angular. This is achieved using the [(ngModel)] syntax, which combines property binding and event binding in a single expression. It allows form inputs to stay in sync with component properties automatically. I used this technique throughout my template-driven forms to connect form fields directly to the component class.

I also learned how to attach event listeners directly in the HTML template using Angular's event binding syntax, allowing me to call class methods and manipulate input values in response to user interactions.

Beyond that, I picked up some additional fundamentals by:

  • Reading the materials provided by the teachers (i.e you).
  • Skimming through the Angular docs.
  • Experimenting with their official tutorials.

It helped reinforce the concepts I was applying and gave me a better understanding of how Angular works overall.
